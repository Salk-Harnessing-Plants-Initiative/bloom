# Planning Docs

Living architecture and roadmap docs for in-flight workstreams. These are
authoritative reading for engineers working on the listed tracks — what's
shipped, what's drafted, what's planned, and who owns each piece.

These docs are written in the first person by their owners, are updated
as work progresses, and are meant to be the **single source of truth** for
their respective tracks.
