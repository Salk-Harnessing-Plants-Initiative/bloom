Starting the Stack

Step 1: Start Development

make dev-up

This command will:

  • Install frontend dependencies
  • Build Docker images
  • Start all services with hot reload
  • Initialize MinIO buckets
  • Apply database migrations

First startup may take 5-10 minutes

Step 2: Verify Services

docker ps

Step 3: Access the Application

Service URL Port
Bloom Frontend http://localhost:3000 3000
Supabase Studio http://localhost:55323 55323
MinIO Console http://localhost:9101 9101

üóÑÔ∏è Database Migrations

Verify Database is Ready

docker exec db-dev pg_isready -U supabase_admin
‚úÖ Output should be: accepting connections

Apply Migrations

make apply-migrations

This command will:

  • Apply all SQL files from supabase/migrations/ in order
  • Skip tables/policies that already exist
  • Show progress for each migration file

Verify Migrations Applied

docker exec db-dev psql -U supabase_admin -d postgres -c "\dt public.*"
